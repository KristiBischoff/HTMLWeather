About the Weather API Project

Have you ever been curious about the ever changing weather? You don't need to be a meteorologist to make weather predictions. Using Python Weather API, we can obtain a plethora of weather data for many geographic locations and various conditions. Rain and snow are not the only game changers when planning a trip, so wind speed and humidity should be accounted for as well. The purpose of this project is to use Python and JSON to request weather data (latitude, longitude, humidity, wind speed , temperature, and cloudiness) for 500 random cities. Scatter plots were created using Matplotlib to visualize the relationship between latitude and:

  • Humidity
  • Cloudiness
  • Temperature
  • Wind Speed
